# The Perfect Season

A starter website for a sports roster-building simulation game.

## Phase 1 Launch
- NBA 82-0
- NFL 17-0
- MLB 162-0
- Free regular season mode
- Casual mode with re-spins
- Ultimate mode with no re-spins
- Leaderboards

## Phase 2
- Paid Championship League
- NBA playoffs
- NFL playoffs and Super Bowl
- MLB postseason and World Series

## Run Locally
npm install
npm run dev
